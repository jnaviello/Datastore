Lesson 1: Understanding Networks

Networks

A group of entities that can communicate even though they are not directly connected. There has to be at least three entities and there has to be a way that entities that are not directly connected can still communicate.

This Video does a good job of explaining it.

Latency

Latency or the time that it takes the message to get from the source to the destination. We can measure latency from the time that you start sending data to the time that the receiver starts receiving it.

This Video gives an overview.

Bandwidth

Bandwidth is the bit-rate of available or consumed information capacity expressed typically in metric multiples of bits per second or bps

This Video gives more detail.

HTML documents follow a tree like structure. There are elements like div,span, h1, h2 etc. which allow programmers to navigate the DOM or document object model. Each one of these elements act like a branch of the tree and follow a parent-child like relationship. As you click on each element you can traverse through the individual branches or from parent to children.

When a programmer talks about DOM they are talking about the tree like structure of a page. This lesson covers this topic in detail.

Avoiding repetition is necessary when programming. You want to avoid having to rewrite the same code multiple times.
